How content flows and moves — the spatial and kinetic language

Layout Philosophy

Current: Editorial Magazine
The Headline

Horizontal rules, alternating L/R layouts, generous whitespace, editorial pacing. Elegant but can feel static.

New: Kinetic Grid
The Headline
Subtext centered below

Centered hero, card-based grid, rounded corners, gradient accents. Dynamic and modular — content can breathe and move.

Layout Principles

Center-First Hero

Hero sections center-aligned with headline, subtext, and dual CTAs stacked vertically. Product screenshot or interactive mockup below. No more left-aligned editorial headers.

Card Grid Over Alternating

Features presented as a card grid (2x2 or 3-col) instead of alternating L/R magazine spreads. Cards have subtle hover states, rounded corners (16px), and elevation on interaction.

Bento Layout for Highlights

Key stats and product capabilities in an asymmetric bento grid (think Apple's feature pages). Different sized cards create visual interest and hierarchy without alternating.

Generous but Tighter

Still generous whitespace, but tightened from the current py-40 to py-24/py-32. Sections feel connected rather than isolated. Content max-width stays at 1280px.

Motion Design

Motion Philosophy: "Purposeful Play"

Every animation should feel like the interface is responding to you, not performing for you. Motion creates continuity between states and draws attention to what matters. We use spring physics (not linear easing) so animations feel natural and alive.

🎯 Entry Animations

  • Hero: Spring-based fade+rise (stiffness: 100, damping: 15). Headline, subtext, CTAs staggered at 100ms.
  • Scroll sections: Fade+rise on viewport entry, 20px translate. Once, with -80px margin trigger.
  • Card grids: Staggered reveal — each card 80ms after the previous. Spring physics, not linear.
  • Counters: Spring-animated number counters for stats (same as current, keep it).

✨ Interaction States

  • Card hover: Subtle lift (translateY -4px) + enhanced shadow + border glow. Spring transition.
  • Button hover: Scale 1.02 + shadow expansion. Gradient buttons shift hue slightly.
  • Link hover: Underline slides in from left (clip-path). Color shift to accent.
  • Tab switch: Indicator slides smoothly (layout animation). Content cross-fades.

🌊 Ambient Motion

  • Hero gradient: Soft radial gradient blob that slowly drifts (CSS animation, 20s loop). Atmospheric, not distracting.
  • Grid pattern: Subtle dot grid background on feature sections. Static, provides texture without movement.
  • Glow effects: Accent-colored glow behind product screenshots. Blurred, semi-transparent.

⚡ Performance Rules

  • Transform only: Animate transform + opacity only. No layout-triggering properties.
  • Reduced motion: Respect prefers-reduced-motion. Instant transitions, no fades.
  • Once animations: Scroll-triggered animations fire once, never replay.
  • Spring config: Default spring { stiffness: 100, damping: 15 }. Snappy but not mechanical.
